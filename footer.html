🌟 製作初衷

回想起剛進入中原大學時，對於廣大的校園、錯綜複雜的公車路線以及各個系辦的位置感到非常困惑。 身為資管系的學姊，我希望透過這次的網頁設計作業，建立一個直觀、好用的「生存指南」，讓學弟妹們能快速找到需要的資訊，不再迷路。

🛠️ 技術成長與 AI 協作

在製作這十個頁面的過程中，我挑戰了許多以前沒嘗試過的網頁技術：

RWD 響應式排版 CSS Flexbox 佈局 JavaScript 即時搜尋過濾 AI 提示工程 (Prompt Engineering) 使用者體驗設計 (UX)
🤖 特別感謝：AI 夥伴 Gemini 的協助

這次專案中最特別的體驗，是全程與 AI 進行協作。透過不斷精煉與 AI 的對話（Prompting）， 我能夠更快速地生成複雜的 CSS 佈局、除錯 JavaScript 程式碼，並將散亂的校園數據整理成好用的查詢系統。 這讓我學會了如何將「人腦的創意」與「AI 的效率」結合，發揮資管人解決問題的最高價值。

💡 學習感悟

這不只是一份網頁作業，更是一次重新認識校園的過程。從整理中原夜市美食、研究證照獎勵辦法，到精確標註每一個系辦的房號， 我學會了如何從海量的資訊中篩選出對使用者最有價值的部分。
